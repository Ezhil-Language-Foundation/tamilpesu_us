tamil.txt2unicode.unicode2encode.unicode2auto(unicode_text, encode_text)[source]¶

This function will convert unicode (first argument) text into other encodes by auto find the encode (from available encodes) by using sample encode text in second argument of this function.

unicode_text : Pass unicode string which has to convert into other encode. encode_text : Pass sample encode string to identify suitable encode for it.

This function tries to identify encode in available encodings. If it finds, then it will convert unicode_text into encode string.
